About the Game

This game takes place in a solar system far far away named "The Malteser Maze" system. Winin this system there are four planets, "Nibble", "Bo", "Kase" and "Magnus". In the game you will play through the first two planets of the system battling against the different aliens in order to repair you rocketship so you can get back home.
